# Portfolio Website

A modern, responsive portfolio website built with React and Vite.

## Features

- Modern and attractive UI design
- Dark/Light theme toggle
- Smooth scrolling navigation
- Responsive design for all devices
- Resume section with education and experience
- Skills showcase with icons
- Projects gallery with hover effects
- Contact form
- Social media integration

## Installation

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Customization

1. Update personal information in all components
2. Replace placeholder images with your own photos
3. Add your resume PDF to the `public` folder
4. Update social media links
5. Add your actual projects with screenshots

## Technologies Used

- React 18
- Vite
- React Icons
- CSS3 with modern features
- Responsive design
